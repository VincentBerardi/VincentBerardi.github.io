Description

Embark on this hyper-casual 3D adventure to explore two levels: the Earth stage and the Moon stage. Find all the star bits in each level to progress and finish the game!

Controls

Input Output
W,A,S,D Move Up, Left, Down, Right
Left-Shift Hold to run
Mouse Look/Turn
Spacebar Jump
Mouse Right-Button Twirl
Esc Pause/In-Game Menu
